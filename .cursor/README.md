# Cursor IDE Configuration

This folder contains configuration files for Cursor IDE to help with development.

## Files

- **rules.md**: Coding rules and guidelines for the project
- **architecture.md**: Detailed architecture documentation
- **mcp.json**: MCP (Model Context Protocol) configuration for AI assistance

## Usage

Cursor will automatically use these files to:
- Provide architecture-aware code suggestions
- Enforce coding standards
- Guide feature implementation
- Review code structure

## MCP Configuration

The `mcp.json` file configures AI assistants to understand:
- Project architecture patterns
- Jet framework usage
- Component development guidelines
- Best practices

## Customization

You can customize these files to match your project's specific needs while maintaining the core Clear Architecture principles.
